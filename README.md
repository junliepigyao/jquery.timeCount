### **简介** 

基于 [damys downCount 项目修改](https://github.com/damys/lab-js/tree/master/downCount) 

### **声明**

本代码是 **damys downCount** 的修改版本 

### **使用**

```
$('.timecount').timeCount({
    date: '1/1/2019 00:00:00',
    countdown: false,
}, function () {
    alert('时间超过当前,请使用倒计时!');
});

```

```
countdown: 默认false，（true为倒计时）
```
